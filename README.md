# 股票数据下载工具

一个基于 Vue 3 的股票数据下载工具，支持 A股、美股、港股的数据下载到 Excel 表格。

## 功能特点

- 📊 支持 A股、美股、港股三大市场
- 📥 一键下载股票数据到 Excel
- 🎨 现代化的 UI 设计
- 🚀 快速响应，实时获取数据

## 使用方法

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. **选择股票市场**：在左侧下拉框中选择 A股、美股 或 港股
2. **输入股票代码**：
   - **A股**：输入6位数字代码，如 `000001`（深市）或 `600000`（沪市）
   - **美股**：输入股票代码，如 `AAPL`、`MSFT`、`TSLA`
   - **港股**：输入5位数字代码，如 `00700`、`09988`
3. **下载数据**：点击"下载数据"按钮，数据将自动下载为 Excel 文件

## API 说明

本项目使用以下免费 API：

- **A股**：新浪财经 API
- **美股**：Yahoo Finance API
- **港股**：新浪财经 API

## 技术栈

- Vue 3
- Vite
- Axios
- XLSX (Excel 导出)

## 注意事项

- 部分 API 可能存在 CORS 限制，如遇问题请检查网络连接
- 股票代码格式请按照各市场要求输入
- 数据获取可能需要几秒钟，请耐心等待
